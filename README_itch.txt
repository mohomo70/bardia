Sword of Xolan-like 2D Action Platformer

CONTROLS:
- Arrow Keys: Move left/right and jump
- Space: Attack with sword

FEATURES:
- Smooth platformer movement with coyote time
- Variable jump height
- Sword combat with hit detection
- Health system with heart collectibles
- Checkpoint system
- Speed-run timer
- Particle effects and juice
- Parallax background

GAME MECHANICS:
- Collect coins by defeating enemies
- Collect hearts to restore health
- Activate checkpoints to save progress
- Try to beat your best time!

TECHNICAL NOTES:
- Built with Phaser 3 and Vite
- Uses placeholder SVG assets
- Responsive design
- Local storage for best times

CREDITS:
- Game engine: Phaser 3
- Build tool: Vite
- Assets: Placeholder SVG graphics
- Inspired by: Sword of Xolan

Enjoy the game!
